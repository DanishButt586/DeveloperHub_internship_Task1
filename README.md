# DeveloperHub Internship Task 1 - MERN eCommerce Platform

## About The Project

This project is a full-stack eCommerce web application built using the MERN stack (MongoDB, Express.js, React, Node.js). It was developed as part of the DeveloperHub Internship program.

The primary purpose of this application is to demonstrate a complete, production-ready shopping experience, from browsing dynamic product catalogs to managing a shopping cart and completing the checkout process.

## Key Features

- **Dynamic Product Catalog**: Browse products loaded dynamically from a MongoDB database, with full support for search, category filtering, and sorting.
- **Product Details**: Dedicated product pages showcasing item specifics and related products.
- **Cart Management**: Add, remove, and update quantities of items in the shopping cart.
- **Checkout & Order Processing**: A complete checkout flow collecting shipping details and processing final orders.
- **Discount & Coupon System**: A server-driven coupon validation system that intelligently applies discounts to the cart total and automatically locks the input fields upon successful application.
- **Responsive UI**: A modern, mobile-friendly interface built with React and Tailwind CSS.
- **Serverless Architecture Ready**: The codebase is cleanly separated into frontend and backend directories, fully optimized for modern cloud deployments.

## Tech Stack

- **Frontend**: React 19, Vite, React Router DOM, Tailwind CSS
- **Backend**: Node.js, Express.js
- **Database**: MongoDB, Mongoose

## Application Structure

The repository is organized into a clean structure:

- /frontend - Contains the React applications, UI components, state management, and frontend build configuration.
- /backend - Contains the Express server, MongoDB models, API routes, and backend business logic.
